Pro add-on for TPF Starter Slider with unlimited sliders, 3D effects, and advanced customization.

== Description ==

TPF Slider Pro extends the free TPF Starter Slider plugin with premium features:

* **Unlimited Sliders** - Create as many sliders as you need
* **3D Coverflow Effect** - Beautiful 3D carousel with depth and perspective
* **3D Flip Transition** - Slides flip with 3D rotation
* **3D Cube Transition** - Cube rotation between slides
* **Text Color Customization** - Full control over title, subtitle, and button colors
* **Priority Support** - Get help when you need it

== Installation ==

1. Make sure TPF Starter Slider (free version) is installed and activated
2. Upload the `tpf-slider-pro` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Pro features will automatically be available in your slider settings
